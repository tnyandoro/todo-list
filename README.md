### Setup

- Go to [todo-list](https://github.com/tnyandoro/todo-list)
- Clone the project to your local machine
- Cd into your directory
- Run npm install
- Type: npm run build
- Go to the 'dist' directory [cd dist]
- Open index.html or run your live server
